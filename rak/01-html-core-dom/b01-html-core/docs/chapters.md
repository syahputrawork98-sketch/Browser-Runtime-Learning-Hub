# Pemetaan Bab B01 - HTML Core

Dokumen ini memetakan bab untuk buku B01 berdasarkan struktur HTML Standard (WHATWG). Fokusnya hanya bagian yang relevan dengan "HTML Core" dan cara kerja browser.

## Jumlah Bab

Total bab yang direncanakan: 10.

Catatan: angka 10 ini adalah jumlah "bab" pada level paling atas (top-level) dari struktur HTML Standard. Di dalam masing-masing bab tersebut ada banyak sub-bab/section yang bisa kita pecah lagi menjadi bab tambahan jika dibutuhkan.

## Daftar Bab (Mengacu ke TOC HTML Standard)

1. C01 - Introduction & Scope
2. C02 - Common Infrastructure
3. C03 - Semantics, Structure, and APIs of HTML Documents
4. C04 - The Elements of HTML
5. C05 - Microdata
6. C06 - User Interaction
7. C07 - Loading Web Pages
8. C08 - Web Application APIs (HTML-defined)
9. C09 - The HTML Syntax (Parsing)
10. C10 - Rendering (HTML-defined)
