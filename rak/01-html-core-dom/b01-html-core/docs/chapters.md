# Pemetaan Bab B01 - HTML Core

Dokumen ini memetakan bab untuk buku B01 berdasarkan struktur HTML Standard (WHATWG).

## Jumlah Bab

Bab top-level bernomor di referensi induk (HTML Standard): 17.

Bab yang kita tulis di buku B01: 17.

## Daftar Bab (Mengacu ke TOC HTML Standard)

Berikut pemetaan bab top-level dari referensi induk ke folder bab di repo.

1. **C01 - Introduction**
- Folder: c01-introduction/
- Fokus: Posisi HTML Standard, ruang lingkup, dan bagaimana spec ini dipakai untuk memahami browser runtime.

2. **C02 - Common Infrastructure**
- Folder: c02-common-infrastructure/
- Fokus: Fondasi umum yang dirujuk lintas bab: terminologi, microsyntaxes, dependencies, dan konsep dasar.

3. **C03 - Semantics, Structure, and APIs of HTML Documents**
- Folder: c03-semantics-structure-apis/
- Fokus: Konsep dokumen HTML, struktur dokumen, dan API dokumen yang membentuk perilaku runtime.

4. **C04 - The Elements of HTML**
- Folder: c04-elements-of-html/
- Fokus: Elemen-elemen HTML dan perilaku runtime-nya, termasuk relasi ke parsing/DOM dan state elemen.

5. **C05 - Microdata**
- Folder: c05-microdata/
- Fokus: Microdata: model data, representasi, dan perilaku terkait di dokumen.

6. **C06 - User Interaction**
- Folder: c06-user-interaction/
- Fokus: Interaksi pengguna dan perilaku runtime: fokus, input, drag-and-drop, dan event terkait.

7. **C07 - Loading Web Pages**
- Folder: c07-loading-web-pages/
- Fokus: Lifecycle loading halaman: navigasi, dokumen baru, dan status pemuatan.

8. **C08 - Web Application APIs**
- Folder: c08-web-application-apis/
- Fokus: API web app yang didefinisikan HTML: timers, messaging, storage hooks, dan API platform terkait.

9. **C09 - Communication**
- Folder: c09-communication/
- Fokus: Komunikasi antar dokumen/konteks: messaging dan mekanisme komunikasi lain yang didefinisikan HTML.

10. **C10 - Web Workers**
- Folder: c10-web-workers/
- Fokus: Workers di HTML Standard: model eksekusi background, messaging, dan hubungan dengan event loop.

11. **C11 - Worklets**
- Folder: c11-worklets/
- Fokus: Worklets: model eksekusi ringan untuk integrasi dengan subsistem tertentu (overview dari sisi HTML Standard).

12. **C12 - Web Storage**
- Folder: c12-web-storage/
- Fokus: Web storage (sessionStorage/localStorage) sebagaimana didefinisikan/dirujuk di HTML Standard.

13. **C13 - The HTML Syntax**
- Folder: c13-html-syntax/
- Fokus: Model parsing HTML: tokenizer, tree builder, error recovery, dan interaksi dengan script/style.

14. **C14 - The XML Syntax**
- Folder: c14-xml-syntax/
- Fokus: Sintaks XML (XHTML) dan perbedaannya dengan HTML parsing dari sisi runtime.

15. **C15 - Rendering**
- Folder: c15-rendering/
- Fokus: Integration points HTML terhadap rendering: render-blocking, default rendering rules, dan kaitan ke CSS.

16. **C16 - Obsolete Features**
- Folder: c16-obsolete-features/
- Fokus: Fitur obsolete: apa yang masih dibahas di spec dan implikasinya untuk implementasi/browser.

17. **C17 - IANA Considerations**
- Folder: c17-iana-considerations/
- Fokus: Pertimbangan IANA (MIME types/registrations) yang relevan untuk HTML dan resource handling.
