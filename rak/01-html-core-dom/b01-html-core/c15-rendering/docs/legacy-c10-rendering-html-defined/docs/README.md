# Docs Bab

Folder `docs/` berisi penjelasan pendukung untuk `README.md` bab ini.

## Isi Yang Cocok Ditulis Di Sini

- Catatan tambahan yang terlalu panjang untuk README utama
- Ringkasan referensi (dengan versi/tanggal dan lokasi section)
- Diagram penjelas non-final
- Checklist pembahasan bab

## Aturan

- Jangan menggandakan isi utama dari `README.md` kecuali untuk konteks singkat.
- Jika merujuk referensi, cantumkan versi/tanggal dan lokasi.
