# Bab: C01 - Alur JavaScript ke Browser

## Ringkasan

Bab ini menjelaskan alur JavaScript dari sumber kode sampai dieksekusi oleh browser, termasuk hubungan antara engine, Web APIs, event loop, dan rendering.

## Tujuan

- Memahami jalur eksekusi JavaScript ke runtime browser
- Menjelaskan komponen utama yang terlibat
- Menjadi dasar untuk bab-bab teknis berikutnya

## Alur Utama

1. Sumber kode JavaScript dimuat oleh browser
2. Engine melakukan parsing dan kompilasi
3. Eksekusi berjalan di call stack
4. Web APIs menjalankan operasi di luar stack
5. Event loop mengatur task/microtask
6. Rendering pipeline memperbarui tampilan

## Istilah Kunci

- Engine
- Call stack
- Web APIs
- Event loop
- Task/Microtask
