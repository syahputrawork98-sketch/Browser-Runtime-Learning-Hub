# Docs Buku

Folder `docs/` berisi dokumen pendukung untuk buku ini (catatan, pemetaan bab, lampiran, dsb).

## Lokasi Bab

Bab berada sejajar dengan `docs/` di root folder buku:

- `c01-.../`, `c02-.../`, `c03-.../`, dst.

Struktur minimal di dalam folder bab:

- `README.md`
- `assets/` (SVG)
- `examples/`
