# Rack Rules: Scheduling Runtime Interaction

Dokumen ini hanya mengatur level rak 06-scheduling-runtime-interaction.

## Wewenang Rak

- Menetapkan batas domain dan istilah kunci rak.
- Menetapkan urutan belajar antar buku dalam rak ini.
- Menetapkan standar kualitas referensi domain ini.

## Batasan Rak

- Tidak mengatur struktur internal buku.
- Tidak memaksa template isi seragam untuk semua buku.
- Tidak mengatur rak lain.

## Aturan Buku di Rak Ini

- Setiap buku harus punya README.md dan folder docs/.
- Setiap buku harus punya docs/book-rules.md dan docs/book-log.md.
- Struktur isi buku boleh berbeda sesuai kebutuhan topik.
- Patokan identitas dan versi buku wajib merujuk ke docs/book-log.md.

## Aturan Log Versi Buku

- Setiap perubahan konten buku ditulis dulu di bagian Perubahan Kerja (Belum Rilis).
- Saat perubahan dianggap cukup, buku membuat rilis versi baru di Riwayat Versi.
- Rak tidak menentukan isi detail versi; keputusan detail versi tetap milik buku.
