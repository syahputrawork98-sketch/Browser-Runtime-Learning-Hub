# Referensi Utama (Buku Besar)

Dokumen berikut dianggap sebagai **buku besar** atau **referensi induk**. Semua bab di repositori ini disusun mengikuti struktur dan isi referensi-referensi ini.

## Standar Inti Browser

- HTML Standard (WHATWG)
- DOM Standard
- CSS Specifications:
  - CSSOM
  - Selectors
  - CSS Cascade
  - Visual Formatting Model (layout)
- WHATWG Fetch Standard
- URL Standard
- Encoding Standard
- Streams Standard
- Web IDL

## Web APIs & Platform

- Service Workers
- Web Workers
- Storage Standard (Storage, Storage Access, Cookies)
- IndexedDB
- Cache Storage
- File API
- WebSocket
- WebRTC
- WebTransport

## Security & Policies

- CORS
- Content Security Policy (CSP)
- Permissions API
- MIME Sniffing Standard

## JavaScript & Engine (Hanya untuk Alur Runtime)

- ECMAScript Specification (ECMA-262)
- JavaScript Engine internals (V8, SpiderMonkey, JavaScriptCore) sebagai referensi implementasi

## Dokumentasi Resmi Browser

- Chrome Platform Status / Dev Rel Notes
- MDN Web Docs (sebagai ringkasan, bukan sumber utama)
- WebKit Release Notes
- Firefox Release Notes

## Catatan

"Buku besar" = referensi induk di atas. Jika ada referensi baru, tambahkan di sini agar semua orang paham sumbernya.
