# Book Rules: Javascript Engine Overview

Dokumen ini hanya mengatur buku 01-javascript-engine-overview.

## Wewenang Buku

- Menentukan cara penyampaian materi buku ini.
- Menentukan struktur bagian yang paling cocok untuk topik ini.
- Menentukan kedalaman penjelasan, analogi, contoh, dan urutan pembahasan.
- Menentukan kapan perubahan siap dirilis menjadi versi baru.

## Batasan Buku

- Tidak mengatur buku lain.
- Tidak mengatur aturan level rak.
- Tidak mengatur aturan level root.

## Kebijakan Penulisan

- Tidak ada template wajib lintas buku.
- Struktur isi boleh berbeda antar buku.
- Buku ini boleh mengubah format internal kapan pun selama log buku tetap konsisten.

## Aturan Log Buku

- Buku ini wajib memiliki log di docs/book-log.md.
- Semua perubahan ditulis dulu di Perubahan Kerja (Belum Rilis).
- Jika perubahan sudah cukup, buat entri versi baru di Riwayat Versi.
- Versi memakai format major.minor.patch.

## Patokan Buku

- Patokan identitas, versi, dan perubahan buku ada di docs/book-log.md.
- README buku boleh ringkas, selama pembaca diarahkan ke log buku.
