# RAK-01: Anatomy & Landscape

Repositori ini membahas sejarah, evolusi, dan landasan filosofis di balik penciptaan browser modern.

## Fokus Pembelajaran
- **Sejarah Browser**: Dari Mosaic hingga Chrome.
- **Evolusi Engine**: Bagaimana V8 mengubah peta persaingan.
- **Mental Model**: Browser sebagai sistem operasi mini.

---
## 📂 Struktur
- **BK-01_Origins**: Sejarah dan visi awal.
- **BK-02_Engine_Evolution**: Transformasi dari interpreter ke JIT.
